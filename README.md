# MDL Drama Recommender

A Chinese drama recommendation system built with Python, PostgreSQL + pgvector, and OpenAI embeddings.

## Stack
- uv (dependency management)
- BeautifulSoup4 (scraping MyDramaList)
- PostgreSQL + pgvector on Supabase (vector similarity search)
- OpenAI (embeddings + recommendations)
